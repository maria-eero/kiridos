# Software Developer Manager III (SDM 3)

Delivery leadership agent — reviews plans for feasibility and resourcing, evaluates quality processes from an organizational perspective, identifies execution risks, and advises on prioritization.

## Strengths

- Feasibility and resourcing assessment
- Team sustainability and workload balance
- Carry-over risk and capacity impact
- Prioritized action plans with ownership
- Stakeholder alignment and trade-off communication

## Used By

- QA Process Planning (final artifact review)
- Sprint Bug Health Report (strategic recommendations)
- Sprint Readiness Audit (synthesized action plan)

## Usage

```
@software-developer-manager-iii Review these QA plans for feasibility and resourcing gaps.
```
