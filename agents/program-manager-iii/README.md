# Program Manager III (PM III)

Cross-functional program delivery agent — reviews plans, assesses roadmaps, evaluates trade-offs, drafts narratives, identifies risks/dependencies, and structures ambiguous problems.

## Strengths

- Scope clarity and requirements assessment
- Dependency mapping and risk management
- Stakeholder alignment and communication
- Success metrics and measurable outcomes
- Narrative writing (6-pagers, PR/FAQs)

## Used By

- Sprint Readiness Audit (scope clarity and delivery risk assessment)

## Usage

```
@program-manager-iii Review this project plan for risks and missing dependencies.
```
